# ROV Arduino Program

Arduino code for the thruster and stepper motor pucks.

## Getting Started

1. Run the test code to know the hardware is working as it should.
2. Test communications are working
3. Program each puck respectively

### Prerequisites

RS485 Library

Arduino NANO Driver

Latest Arduino IDE
